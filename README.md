# Reno Notice Board

A full-stack Notice Board application built using Next.js, Prisma, PostgreSQL (Neon), and Tailwind CSS.

## Features

- Create Notice
- View Notices
- Update Notice
- Delete Notice
- Search Notices
- Category Filter
- Priority Support
- PostgreSQL Database
- Prisma ORM
- REST API

## Tech Stack

- Next.js
- React
- Tailwind CSS
- Prisma
- PostgreSQL (Neon)

## Installation

```bash
npm install
npm run dev
```
